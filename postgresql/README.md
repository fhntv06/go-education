Этот модуль представляет собой pet-project который направлен на работу c CRUD

### В модуле реализовано:
- модель
- хранилище
- взаимодействие с базой данных
- миграция
- весь CRUD

### Стек
- Язык программирования - Go
- СУБД PostgreSQL
